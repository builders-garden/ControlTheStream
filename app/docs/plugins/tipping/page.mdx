# Tipping

Enable your audience to support you directly. Viewers can send **USDC on Base** along with a custom text message.
The overlay will display the tip amount and the message in real-time.

## How to Set It Up

  src="/docs/images/plugins/tips/how-to-setup.png"

1. Navigate to the **Tips** plugin in your dashboard and click on the edit button
2. Set a valid basename or a valid address where you want to receive the tips
3. Click on the confirm button

## Custom Messages

> **Note:** Custom text messages are available only for tips over $5.
