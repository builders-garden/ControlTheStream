# Token Highlight

Feature specific assets for your community. You can highlight up to **6 different tokens** to be displayed within the mini-app.

1. Navigate to the **Token Highlight** plugin in your dashboard
2. Click the "Add more tokens" button to start searching for a new token to add
3. Search on the chain you prefer the tokens you want to add, select them and click confirm
4. Hide or remove a token from the list by clicking on the corresponding button

- In the desktop version of the app, if the user clicks on a featured token, they will be redirected to the token's page on [Matcha](https://matcha.xyz/)
- From the mini app, the user will have the option to see the chart of the token or to start the buying process. If they click on the "trade" button, the integrated wallet will be opened with the featured token pre-set as the token that is being bought.
