# Creator Coin

Promote your personal brand or favorite creator. This plugin allows you to highlight a single **Zora Creator Token**.

## How It Works

1. Navigate to the Creator Coin plugin in your dashboard
2. Enter the Zora Creator Token you want to feature
3. The token will be displayed for your viewers to interact with

## Features

- Viewers can view the token details
- Direct trading capability
- Perfect for promoting creator economy
- Build community around your creator token
