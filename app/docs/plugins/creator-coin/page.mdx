# Creator Coin

Promote your personal brand or favorite creator. This plugin allows you to highlight a single **Zora Creator Token**.

1. Navigate to the **Creator Coin** plugin in your dashboard
2. Click the "Add coin" button to start searching for a new creator coin to add
3. Search on **Base** blockchain the token you want to add, select it and click confirm
4. Remove the token by clicking on the corresponding button

- From the mini app, the user will have the option to see the chart of the coin or to start the buying process. If they click on the "trade" button, the integrated wallet will be opened with the token pre-set as the one that is being bought.
