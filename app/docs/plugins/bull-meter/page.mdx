# Bull Meter (Polls)

Create interactive, skin-in-the-game polls. Viewers vote by committing **1 cent of USDC on Base**. The overlay displays live voting, and once the timer expires, the final result is automatically announced to the stream.

## How It Works

1. Create a poll from your dashboard
2. Viewers commit 1 cent USDC to vote
3. Live voting is displayed on your overlay
4. Results are announced automatically when timer expires

## Features

- Real-time vote tracking
- Automatic result announcement
- Skin-in-the-game engagement
- Timer-based voting periods
