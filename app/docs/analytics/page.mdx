# Analytics

The **Analytics** tab in your dashboard provides a comprehensive breakdown of how your audience is engaging with your onchain features.
Use these insights to understand what content drives the most interaction and revenue.

## Available Metrics

### Tips

Monitor your earnings with data on:

| Metric              | Description                               |
| ------------------- | ----------------------------------------- |
| Total Tips Received | Number of tips sent by viewers            |
| Unique Tippers      | Number of unique wallets that have tipped |
| Total Amount (USDC) | Total value collected in tips             |

You can also view a full chronological list of every tip sent.

### Polls

Gauge community sentiment by tracking:

| Metric              | Description                              |
| ------------------- | ---------------------------------------- |
| Total Votes         | Number of votes cast in Bull Meter polls |
| Unique Voters       | Number of unique wallets that voted      |
| Total Amount (USDC) | Total value committed to polls           |

Includes a detailed list of all individual votes.

### Creator Coins

See how much interest your creator token is generating:

| Metric       | Description                                 |
| ------------ | ------------------------------------------- |
| Total Opens  | Number of times the creator coin was viewed |
| Unique Opens | Number of unique viewers                    |

Includes a list of specific viewers who engaged with the coin.

### Featured Tokens

Track the performance of your highlighted assets:

| Metric       | Description                                 |
| ------------ | ------------------------------------------- |
| Total Opens  | Number of times featured tokens were viewed |
| Unique Opens | Number of unique viewers                    |

See which tokens your community is most interested in trading.

> **Tip:** Use these analytics to optimize your content strategy and understand which features resonate most with your audience.
